Sorting Hat

This program efficiently assigns first-year Lewis & Clark students to their first-year seminar courses based on their class preferences. This program also ensures that classes gender balances are within a reasonable threshold of the overall student population in addition to ensuring that each class does not have too many athletes.

  1. Choose Students File

  2. Choose Sections File

  3. Set Gender Ratios
    100%
    100%

    These sliders set the maximum percentage of male or female students allowed in any given section/class. Setting the slider to 70% for female students would mean that any section should be composed of at most 70% female students.

  4. Set Athlete Ratio 100%

    These sliders set the maximum percentage of student-athletes allowed in any given section/class.

  5. Run the Program
  6. Save the Results to File
